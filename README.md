# MediFlow - Hospital Wait Time Management System

## What is this project?

MediFlow helps patients check hospital wait times and book appointments online. It also helps hospital staff manage patient flow efficiently.

## Why this project?

Long hospital wait times are a real problem. Patients don't know when to come, hospitals don't know how busy they'll be. This project aims to solve that.

## Features (Planned)

- User registration and login (Patients, Hospital Staff)
- View real-time wait times for different hospitals
- Book appointments online
- Receive notifications about appointment status
- Hospital dashboard to manage patient flow

## Tech Stack

- **Frontend**: React.js (JavaScript library for user interfaces)
- **Backend**: Node.js + Express (JavaScript on the server)
- **Database**: MongoDB (stores user data, appointments)
- **Real-time updates**: Socket.io (live wait time updates)
